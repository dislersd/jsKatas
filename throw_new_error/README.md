## Task

Implement a function named isPositive that takes a single number _n_ as input. If the number is positive return "YES", if a number is 0 throw an error with message "Zero Error", if a number is negative throw an error with message "Negative Error"

### Input Format

single Number _a_

### Output Format

Function must return "YES" or an error message

### Sample Input
3
6
0
-1
### Sample Output
"YES"\
"YES"\
"Zero Error"\
"Negative Error"
### Explanation
3 is positive so we log "YES"\
6 is positive so we log "YES"\
0 so we log "Zero Error"\
-1 is negative so we log "Negative Error"
